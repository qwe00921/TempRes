;---------------------------------------------------------------
; 子弹参数配置（关键字全小写）
; ani				子弹特效
; shoot_ani			发射时额外播个特效
; shoot_sound		发射时额外播个声音
; fly_sound			飞行过程中的声音
; fly_action		飞行动作特效
; explode_ani		爆炸特效
; explode_sound		爆炸声音
;
; width				子弹资源有效宽度（不是图片宽度）
; sub_distance		距离扣除数
;---------------------------------------------------------------
